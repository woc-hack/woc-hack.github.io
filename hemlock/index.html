Projects with known vulnerabilities that have been fixed

  • RIOT is a real-time multi-threading operating system.
    RIOT is subject to CVE-2017-8289.
    The vulnerability was fixed by commit 92fe84214967012e951f7aee5bcee9977c426eef as described at github.com/RIOT-OS/RIOT/pull/6961
    Full Report

  • QEMU is a generic and open source machine & userspace emulator and virtualizer.
    QEMU is subject to a buffer overflow vulnerability as described in CVE-2018-17962.
    The fix is described here.
    The vulnerability was fixed by commit b1d80d12c5f7ff081bb80ab4f4241d4248691192.
    Full Report

  • LZ4 is a widely-used lossless compression algorithm.
    LZ4 is subject to a heap-based buffer overflow in releases prior to 1.9.2 as described in CVE-2019-17543.
    The vulnerability was fixed by commit d7cad81093cd805110291f84d64d385557d0ffba.
    Full Report
